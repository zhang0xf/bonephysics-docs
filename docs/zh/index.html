欢迎使用 Bone Physics 文档

Bone Physics 是一款 Blender 插件，用于在风格化角色动画中为头发、飘带、尾巴等动态元素创建基于刚体的骨骼物理模拟。

本插件基于 blender_mmd_tools 开发，目的是将 MMD 的物理系统移植到 Blender 中，并作为一个独立模块使用，从而兼容 Blender 内置的 Rigify 骨架以及其他常见绑定系统。

本插件已在 GitHub 上开源，并按照 GPL-3.0 许可证发布。

由于本插件基于 blender_mmd_tools 开发，在设置物理参数时，你可以参考许多现有 MMD 模型中的物理配置。

视频教程以及完整的工作流程演示可以在 YouTube 和 Bilibili 上查看。
