问答

问：我如何获取插件？

你可以直接从 GitHub Releases 页面下载插件的 .zip 文件。

问：插件支持哪些版本的 Blender？

插件 最低支持版本 为 Blender 4.2 LTS。

该插件主要在 Blender 4.2 上开发和测试。更高版本的 Blender（如 Blender 4.5）也可能可用，但尚未经过完整测试。

问：插件使用什么许可证？

该插件采用 GPL-3.0 许可证发布。你可以自由使用、修改和重新分发插件，但需遵守 GPL-3.0 许可证条款。

问：如何获取 MMD 插件？

可以从 MMD Tools 下载 MMD Tools 插件。

问：如何获取 MMD 模型以供参考？

你可以在网上找到许多 MMD 模型。

例如，你可以访问 Bilibili 网站，搜索“绝区零UP主激励计划”。在对应页面找到 模型下载 区域，点击 下载模型，即可获取游戏 绝区零 的最新 MMD 角色模型。

问：我可以在哪里提交 Bug 或请求新功能？

你可以通过 GitHub Issues 提交 Bug 和功能请求。
